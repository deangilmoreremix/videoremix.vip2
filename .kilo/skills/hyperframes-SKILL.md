# HyperFrames

Open-source video rendering framework that lets you write HTML and render videos, built for AI agents.

## What it does

HyperFrames is a complete video creation toolkit that allows you to:
- Write HTML compositions with animations
- Preview videos in real-time
- Render high-quality MP4 videos
- Use various animation libraries (GSAP, Anime.js, CSS, Lottie, Three.js)
- Process media assets (TTS, transcription, background removal)

## Skills included

This repository provides 13 specialized skills:

### Core Framework
- **hyperframes** - Main framework for creating video compositions
- **hyperframes-cli** - Command-line tools for development workflow
- **hyperframes-registry** - Component and block registry

### Animation Libraries
- **gsap** - GSAP animation library integration
- **animejs** - Anime.js animation library
- **css-animations** - Pure CSS animations
- **waapi** - Web Animations API
- **lottie** - Lottie animation files

### 3D & Advanced
- **three** - Three.js 3D graphics
- **tailwind** - Tailwind CSS integration

### Utilities
- **hyperframes-media** - Media processing (TTS, transcription, image editing)
- **remotion-to-hyperframes** - Migration guide from Remotion
- **website-to-hyperframes** - Convert websites to video compositions

## Quick start

1. Install the skills: `npx skills add heygen-com/hyperframes`
2. Create a composition: `/hyperframes create my-video`
3. Preview: `/hyperframes-cli preview`
4. Render: `/hyperframes-cli render`
