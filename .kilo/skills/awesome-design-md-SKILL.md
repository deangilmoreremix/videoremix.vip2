# Awesome DESIGN.md

Curated collection of DESIGN.md files from real websites for pixel-perfect UI generation.

## What it does

Provides 73+ DESIGN.md files extracted from popular websites like Airbnb, Apple, BMW, Claude, Vercel, and many more. Each DESIGN.md file contains detailed design specifications that AI agents can use to generate UI that matches the visual style of these websites.

## How to use

1. Copy any DESIGN.md file from the `design-md/` directory to your project root
2. Tell your AI agent: "Build me a page that looks like this DESIGN.md"
3. Get pixel-perfect UI that matches the specified design system

## Available designs

The repository includes DESIGN.md files for:
- **Tech companies**: Vercel, Netlify, Stripe, GitHub, Figma
- **AI platforms**: Claude, OpenAI, Anthropic, Cohere
- **Design tools**: Figma, Framer, Webflow
- **Productivity**: Notion, Airtable, Linear
- **Finance**: Stripe, Coinbase, Binance
- **Automotive**: BMW, Tesla, Bugatti
- **Consumer**: Airbnb, Spotify, Netflix
