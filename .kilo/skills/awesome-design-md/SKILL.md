# Awesome DESIGN.md

A curated collection of DESIGN.md files inspired by popular brand design systems. Drop one into your project and let coding agents generate matching UI.

## What is DESIGN.md?

[DESIGN.md](https://stitch.withgoogle.com/docs/design-md/overview/) is a plain-text design system document that AI agents read to generate consistent UI.

| File | Who reads it | What it defines |
|------|-------------|-----------------|
| `AGENTS.md` | Coding agents | How to build the project |
| `DESIGN.md` | Design agents | How the project should look and feel |

## How to Use

1. Copy a site's `DESIGN.md` into your project root
2. Tell your AI agent to use it
