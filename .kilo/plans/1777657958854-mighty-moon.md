## 🎯 Mission Objective

Create rich, engaging modal popups for dashboard app cards featuring:
- **Large AI-generated thumbnails** (DALL-E 3, hosted on Supabase)
- **Comprehensive product descriptions** (long-form)
- **Monetization strategies** (for local businesses & individuals)
- **Superpowers Skills methodology content** (GTM tonalities)
- **Professional design** aligned with "Functional Elegance" philosophy

These popups will replace the current SalesDropdown inline accordion and provide a complete sales experience that drives conversions.

---

## 📋 Executive Summary

**Current State:**
- Dashboard cards use inline `SalesDropdown` with 3 brief sections
- AI thumbnails exist but aren't all displayed (integration fix applied, needs verification)
- Content is concise, not long-form persuasive copy

**Desired State:**
- Clicking an app card opens a fullscreen/near-fullscreen modal popup
- Hero section with large AI thumbnail + app name + tagline
- Multi-section sales page layout using GTM Skills tonalities
- Extended "How to Profit" sections for local businesses & individuals
- Clear call-to-action (Purchase or Open App)
- Mobile-optimized, accessible, animated

**Content Strategy (NEW):** Concise, LLM-generated copy (50-150 words per section) using GTM Skills tonalities, not exhaustive long-form. Prioritize clarity and persuasion over length.

```
    │   │
```

**Trigger Behavior:**
- Single click anywhere on card opens ProductDetailModal
- Previous SalesDropdown removed from card
- Modal replaces card-centric interaction for owned apps
- Non-owned apps show PurchaseModal inside ProductDetailModal

## 📊 Data Structure Enhancement

### Extended SalesCopy (Concise LLM-Friendly Format)

```typescript
interface ExtendedSalesCopy {
  // Identity
  tonality: string;          // e.g., "Steve Jobs"
  tagline: string;           // 5-12 words hook
  summary: string;           // 1-2 sentences elevator pitch
  
  // Core content (all 50-100 words each)
  whatItDoes: string;        // Single paragraph, capabilities embedded
  howItWorks: string;        // 3-5 numbered steps integrated into narrative
  howToProfit: {
    forLocalBusinesses: string;  // 3-4 sentences: model + pricing + target + revenue
    forIndividuals: string;      // 3-4 sentences: service idea + platforms + income potential
  };
  whyYouNeedIt: string;     // 2-3 paragraphs: pain + value + urgency (tonality-infused)
  
  // Supporting (concise)
  useCases: Array<{industry: string; scenario: string; outcome: string}>; // 3 items max
  testimonials: Array<{
    quote: string;          // 1-2 sentences
    name: string;
    role: string;
    businessType: string;
    rating: number;
  }>;                       // 2-3 testimonials max
  
  // Optional
  icon?: string;            // Lucide icon name for section headers
}
```
**Design Principle:** Keep it skimmable. Users scan, don't read novel-length copy. Every word must earn its place.

## 📝 Content Strategy (Superpowers Skills Methodology + LLM Generation)

### Philosophy
- **Concise first**: 50-100 words per core section (not 300+)
- **Tonality-infused**: Every section carries the assigned GTM Skills voice
- **Actionable**: "How to Profit" sections give specific, immediate monetization paths
- **Scannable**: Bullets, short paragraphs, clear headings
- **LLM-generated at build time**: Use prompts to produce consistent, on-brand copy for all 134 apps, with human review spot-checks

### GTM Skills Tonality Assignment Matrix

Each app's sales copy will reflect one of these expert voices (concise versions):

**12 Core Tonalities:**
1. **Steve Jobs** – Reality distortion field. "Think different." Revolutionary simplicity.
2. **Seth Godin** – Remarkable, shareworthy. "Purple Cow." Story-driven.
3. **Chris Voss** – Tactical empathy. "Mirroring." Psychological precision.
4. **Jeff Bezos** – Customer obsession. "Day 1 thinking." Long-term strategic.
5. **Challenger Sale** – Insight-led disruption. "Teach, tailor, take control."
6. **Trusted Advisor** – Relationship-first. "Trust before transaction."
7. **Cormac McCarthy** – Sparse, visionary. "The emptiness speaks."
8. **Hemingway** – Direct, clear. "Write hard and clear about what hurts."
9. **Value-Based** – ROI quantification. "$4 return for every $1 spent."
10. **Pain Point Research** – Diagnostic discovery. "What's your bottleneck?"
11. **David Ogilvy** – Benefit-driven. "The consumer isn't a moron."
12. **Growth Hacker** – Scalable experiments. "Growth loops, not campaigns."

**Category → Tonality Distribution** (rotate within category for variety):
```
Video:         [Steve Jobs, Hemingway, Growth Hacker]
AI-Image:      [Seth Godin, Cormac McCarthy, Visionary]
Creative:      [Seth Godin, David Ogilvy, Innovative]
Lead-Gen:      [Challenger Sale, Value-Based, Trusted Advisor]
Personalizer:  [Chris Voss, Pain Point Research, Socratic]
Branding:      [Jeff Bezos, Trusted Advisor, Strategic]
```

**Tonality Markers (Styling in Modal):**
- Taglines use tonality's signature word choice (Jobs: "revolutionary", Godin: "remarkable", Voss: "calibrate")
- Sentence rhythm varies: Hemingway = short sentences; McCarthy = long, sparse; Ogilvy = benefit-first
- Icons/symbols per tonality (optional: small badge in header showing tonality)

**Prompt Template (per app):**
```

App description: {description}
Category: {category}
GTM Skills Tonality: {tonality} (write entirely in this voice)

Write concise, scannable copy. Each section below should be 2-4 sentences max unless noted.

OUTPUT ONLY VALID JSON matching this schema:
{
  "tagline": "5-12 word hook (compelling, benefit-driven)",
  "summary": "1-2 sentence elevator pitch",
  "whatItDoes": "50-80 words explaining core functionality",
  "howItWorks": "Brief narrative: first [step], then [step], finally [step]. 3-5 steps, 60-100 words total.",
  "howToProfit": {
    "forLocalBusinesses": "What service model? Who pays? What price range? Potential monthly revenue. 3-4 sentences.",
    "forIndividuals": "What side-hustle service to offer? Which platforms (Fiverr/Upwork/local)? How much time? Income potential. 3-4 sentences."
  },
  "whyYouNeedIt": "Start with the pain point (1 sentence). Then the value proposition in {tonality} voice (2 sentences). End with urgency/trend (1 sentence). Total 3-4 sentences.",
  "useCases": [
    {"industry":"...", "scenario":"...", "outcome":"..."}
  ],
  "testimonials": [
    {"quote":"1-2 sentence realistic quote","name":"...","role":"...","businessType":"...","rating":5}
  ]
}

Tonality guidelines:
- {tonality}: Use {toneSpecificGuidance} (max 3 bullet points of voice traits)
- Be concise but persuasive. Every word earns its place.

Now output ONLY the JSON object:
```
