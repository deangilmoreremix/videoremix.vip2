## Stripe CLI Commands

```bash
# Authenticate
stripe login

# Create bundle product (example for first bundle)
stripe products create \
  --name="Sales, Lead Gen & Prospecting Bundle" \
  --description="10 AI-powered sales and lead generation tools" \
  --metadata=bundle_id,sales-lead-gen-bundle,bundle_price,39700

# Create price for bundle
stripe prices create \
  --product=<product_id> \
  --unit-amount=39700 \
  --currency=usd \
  --metadata=type,one_time,billing_cycle,lifetime

# Run sync script
npx tsx scripts/sync-stripe-products.ts
```
