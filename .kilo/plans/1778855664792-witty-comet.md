# Plan: Merge Original Landing Page Content into Premium Landing Page

## Problem
The premium landing page (`src/components/premium/LandingPage.tsx`) is a redesigned "impeccable" page that replaced the original landing page (`src/pages/LandingPage.tsx`) but is missing many sections and content from the original. The app currently routes to the original `src/pages/LandingPage.tsx`, but the user wants all original content included in the new premium design.

## Analysis

### Original Landing Page (`src/pages/LandingPage.tsx`) has these sections NOT in the premium page:
1. **SpecialHero** - Hero section with video preview, key benefits, countdown timer, active users
2. **ProblemSection** - Problem/pain points section
3. **SolutionSection** - Solution explanation section
4. **FeatureMap** - Interactive feature map with title/subtitle props
5. **PersonalizationWorkflowSection** - Workflow visualization
6. **ToolsCarouselSection** - Carousel of personalization tools
7. **BenefitsSection** - Benefits with data from LandingPageContext
8. **AppGallerySection** - Gallery of 116+ apps
9. **DemoSection** - Product demo
10. **CaseStudiesSection** - Case studies
11. **TestimonialsSection** - Testimonials from LandingPageContext
12. **PricingSection** - Pricing from LandingPageContext
13. **GuaranteeSection** - Money-back guarantee
14. **FAQSection** - FAQs from LandingPageContext
15. **FinalCTA** - Final call-to-action
16. **SEO** - SEO meta tags component
17. **Hero Stats Section** - "Trusted by Marketing Teams Worldwide" with 3 stats
18. **Marketing Impact Stats** - 4 stats section
19. **Interactive Comparison Table** (original version with 5 rows)
20. **Logo Wall** (original with real company logos: Netflix, Adobe, Spotify, Airbnb, Meta, Stripe)
21. **Floating Icons** (3 floating icons)
22. **Live Activity Feed** (fixed position)

### Premium Landing Page (`src/components/premium/LandingPage.tsx`) has these unique sections NOT in original:
1. **ParallaxSection** wrapper for Before/After
2. **PersonalizationSimulator** - Interactive simulator
3. **Deep Benefits** - 4 alternating image+text benefit blocks
4. **Scannable Features** - 4-icon feature grid
5. **Mid-Page CTA** - "Ready to Transform Your Marketing?"
6. **What's Included** - 3-item feature grid
7. **UGC Strip** - Stats strip (4 stats)
8. **FAQ** - 3 FAQ items (hardcoded, not from context)
9. **Visuals (Demo)** - Image-based demo section
10. Uses `react-helmet-async` instead of custom SEO component
11. Uses `container-max section-padding` CSS classes
12. Has hardcoded testimonials (not from LandingPageContext)

## Approach
Update `src/components/premium/LandingPage.tsx` to include ALL sections from the original landing page while keeping the premium design's unique sections and styling. The premium page should be the single source of truth going forward.
