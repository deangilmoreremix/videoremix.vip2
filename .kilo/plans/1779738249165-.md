# Fix Missing ResultGrid Component Import

## Goal
Fix build error caused by incorrect import path for `ResultGrid` component.

## Problem
- `AiGameDesignAgentTeamPage.tsx:12` imports `ResultGrid` from `@/components/agent-ui/ResultGrid`
- The file `src/components/agent-ui/ResultGrid.tsx` does not exist
- `ResultGrid` is actually defined in `src/components/agent-ui/ResultCard.tsx` (lines 86-98)
- Other pages correctly import both `ResultCard` and `ResultGrid` from `ResultCard.tsx`

## Solution
Changed the import in 5 files from:
```typescript
import { ResultCard } from "@/components/agent-ui/ResultCard";
import { ResultGrid } from "@/components/agent-ui/ResultGrid";
```
to:
```typescript
import { ResultCard, ResultGrid } from "@/components/agent-ui/ResultCard";
```
