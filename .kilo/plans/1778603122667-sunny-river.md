# Plan: Combine Onboarding and Guided Setup Wizards

## Problem Statement
Currently there are two separate wizard experiences with fragmented UX:

**Onboarding Wizard Steps (4 steps after fix):**
1. **Select Goals** - GoalSelectionStep.tsx (Build, Launch, Grow, Automate)
2. **Choose Niche** - NicheSelectionStep.tsx (Coach, E-Commerce, SaaS, Agency, Creator, Local Business, B2B, Non-Profit, Education, Real Estate, Fitness)
3. **Select Categories** - CategorySelectionStep.tsx (app categories pre-recommended based on goals+niche)
4. **Recommendations** - RecommendationsStep.tsx (shows apps filtered by selected categories)

**Guided Setup Tasks (shown separately in GuidedSetupStep.tsx):**
1. **Complete Your Profile** - Edit profile settings
2. **Get Your First App** - Browse and purchase first tool  
3. **Create Your First Video** - Start creating content
4. **Explore Multiple Apps** - Try out 5 different tools
5. **Customize Your Dashboard** - Set theme and layout preferences

The goal is to:
- Combine ALL steps into a **single streamlined wizard** with 4 steps total
- Final screen (**Your Personalized Toolkit**) shows recommended apps + condensed guided setup tasks
- Use the premium design of the OnboardingWizard (ParticleBackground, GradientOrb, coral theme #cc785c, smooth animations)
- Present apps in an engaging, personalized manner with ownership status and actions

## Current Architecture Analysis

### Onboarding Wizard Steps (4 steps in current code)
- **Step 1**: `GoalSelectionStep.tsx` - Select goals (Build, Launch, Grow, Automate)
- **Step 2**: `NicheSelectionStep.tsx` - Choose niche (11 options: Coach, E-Commerce, SaaS, etc.)
- **Step 3**: `CategorySelectionStep.tsx` - Select categories (pre-recommended based on goals+niche)
- **Step 4**: `RecommendationsStep.tsx` - Shows apps filtered by selected categories

### Guided Setup Tasks (separate screen in `GuidedSetupStep.tsx`)
1. **Complete Your Profile** (User icon) - links to /dashboard#profile
2. **Get Your First App** (ShoppingCart icon) - links to /#tools
3. **Create Your First Video** (Video icon) - links to /dashboard#create
4. **Explore Multiple Apps** (Compass icon) - links to /#tools
5. **Customize Your Dashboard** (Settings icon) - links to /dashboard#preferences

### Data Flow
```
User selections (goals + niche) → getRecommendedCategories() → CategorySelectionStep pre-selects
→ RecommendationsStep filters appsData by selectedCategories → ToolkitStep shows apps + setup tasks
```

### Key Issues
1. **Two separate wizards** create fragmented UX - user must complete onboarding, then separately do guided setup
2. **Step 4 (RecommendationsStep)** shows apps but no setup guidance
3. **GuidedSetupStep** is disconnected from personalized recommendations
4. **No clear action path** - users see apps but aren't guided toward next actions
5. **Duplicate progress indicator** - 4 steps feels long for essentially 3 decision steps + 1 results screen

## Proposed Solution

### New Wizard Structure (4 Steps - Combined)
All steps use the premium design: coral theme (#cc785c), ParticleBackground, GradientOrb, framer-motion animations.

1. **Select Goals** (from Onboarding Wizard - unchanged)
   - UI: GoalSelectionStep.tsx with coral/blue selection state
   
2. **Choose Your Niche** (from Onboarding Wizard - unchanged)
   - UI: NicheSelectionStep.tsx
   
3. **Pick Your Focus Areas** (from Onboarding Wizard - enhanced)
   - UI: CategorySelectionStep.tsx with pre-recommended categories
   
4. **Your Personalized Toolkit** (NEW - merges Recommendations + Guided Setup)
   - Shows recommended apps based on selections (from RecommendationsStep)
   - Includes condensed guided setup tasks (from GuidedSetupStep):
     * Complete Your Profile
     * Get Your First App
     * Create Your First Video
     * Explore Multiple Apps
     * Customize Your Dashboard
   - Single screen with app cards + checklist

### Step 4 Redesign: "Your Personalized Toolkit"
Replace both RecommendationsStep and GuidedSetupStep with a unified screen:
- **Header**: "Your Personalized Toolkit" or "Apps We Recommend For You"
- **Subtext**: "Based on your goals and niche, here are your handpicked tools"
- **App Showcase**: Category sections with apps (like current RecommendationsStep)
- **Enhanced App Cards**: 
  - Show app name, description, icon (from LazyIcon)
  - Show status: Owned / Not Purchased (with lock overlay)
  - Action buttons: "Open" if owned, "Purchase" or "Learn More" if not
  - Quick stats or key benefit highlight
- **Quick Setup Tasks Section** (condensed from GuidedSetupStep):
  - "Complete Profile" (if not done)
  - "Set Up Payment" (if never purchased)
  - "First App Tutorial" (link to getting started)
  - Show as compact progress items at bottom or top
- **Call-to-Action Button**: "Continue Exploring" or "Get Started with These Tools"
- **Skip/Wizard completion**: Button to finish onboarding and go to dashboard

## Decisions (Based on User Request)
1. **GuidedSetupStep tasks will be integrated** into the final toolkit screen as a condensed "Getting Started" checklist (not removed entirely)
2. **After completing the wizard**, users will be navigated to the dashboard (as currently implemented in `completeOnboarding`)
3. **Back navigation** will be kept on the final screen to allow reviewing choices
4. **Final step will have explicit "Complete Onboarding" button** that calls `completeOnboarding()` and navigates to dashboard
5. **Design will follow the premium style** of the OnboardingWizard container (coral theme #cc785c, ParticleBackground, GradientOrb) with card styling inspired by GoalSelectionStep but using coral accents instead of blue
