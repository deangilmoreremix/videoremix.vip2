# Personalization Module User Walkthrough

This walkthrough shows how to use the new personalization flow.

## 1. Start a prospect scan

- Open the personalization widget.
- Enter the prospect username.
- Select the target VideoRemix app and mode.
- Click `Scan Prospect`.

## 2. Review profiles

- The scan returns a list of discovered profiles.
- Review the platform, profile URL, and confidence scores.

## 3. Generate personalization

- Click `Generate Personalization`.
- The system uses the profile data to generate outputs such as video scripts, email drafts, and proposals.

## 4. Save results

- Click `Save Results` to persist the scan and generated outputs into Supabase.
- Use the saved content for follow-up campaigns, creative scripts, or client proposals.
